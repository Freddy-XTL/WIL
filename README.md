# NovaTech Digital

NovaTech Digital is a web application that provides a platform for users to register, log in, and access various features and services.

## Installation

To run the NovaTech Digital application, follow these steps:

1. Clone the repository:
```
git clone https://github.com/your-username/novatech-digital.git
```
2. Navigate to the project directory:
```
cd novatech-digital
```
3. Open the `index.html` file in your preferred web browser.

## Usage

The NovaTech Digital application provides the following features:

1. **Home Page**: The home page allows users to register or log in to the application.
2. **Help Page**: The help page provides contact information for users to reach out to the NovaTech Digital team.
3. **Login Page**: The login page allows registered users to log in to the application.
4. **Register Page**: The register page allows new users to create an account with NovaTech Digital.

## API

The NovaTech Digital application does not currently provide any public API.
